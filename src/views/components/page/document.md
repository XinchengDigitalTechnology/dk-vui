# Page 页面组件

- 基础页面为列表页，页面高度100%，溢出隐藏
- edit为true时为编辑页，横向溢出隐藏，纵向溢出为自动
- 编辑页保留滚动位置

## Attributes

| 属性名 | 说明|  类型 | 默认值 |
| --- | --- | --- | --- | 
| edit | 是否为编辑页 |boolean |—|
| footer | 为编辑页时底部固定区域配置 { offset } |object |{}|

## Slots

| 插槽名 | 说明|
| --- | --- |
| left | 左侧区域 |
| footer | 编辑页底部固定区域 |

## 按钮

```js
<VPage>页面内容</VPage>
```
