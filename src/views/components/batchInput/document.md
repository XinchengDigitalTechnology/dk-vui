## 使用

```js
<VBatchInput v-model="value" />
```

## Attributes

| 属性名 | 说明                       | 类型   | 默认值 |
| ------ | -------------------------- | ------ | ------ |
| modelValue   | 绑定值 | string | —      |
| maxlength   | 同原生 maxlength 属性 | string / number | 200      |

## Events

| 事件名  | 说明           |
| ------- | -------------- |
| search | 点击搜索按钮时触发 |

## Slots

| 插槽名  | 说明           |
| ------- | -------------- |
| default | 弹窗内操作区域插槽，会加在清空按钮后面 |
