# Auth 权限组件

- 区域权限组件

## Attributes

| 属性名 | 说明                       | 类型   | 默认值 |
| ------ | -------------------------- | ------ | ------ |
| auth   | 绑定值，必须与权限路径一致 | string | —      |

## Slots

| 插槽名  | 说明           |
| ------- | -------------- |
| default | 自定义默认内容 |

## 使用

```js
<VAuth auth='/pms/order/add'>区域权限</VAuth>
```
